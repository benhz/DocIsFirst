# 开发文档

欢迎来到 Qicat 开发文档区域！

## 概述

本文档区域包含了项目开发的各种技术指南和最佳实践，帮助开发者快速上手并掌握项目开发技能。

## 文档导航

### 后端开发

- **Node.js 开发指南**：学习如何使用 Node.js 进行后端开发
- **数据库设计**：数据库设计原则和最佳实践
- **API 开发**：RESTful API 和 GraphQL 的设计与实现

### 前端开发

- **React 开发指南**：React 应用开发的核心概念
- **组件开发**：如何开发可复用的组件
- **样式开发**：CSS-in-JS 和主题定制

## 快速开始

如果你是第一次接触本项目，建议按以下顺序阅读：

1. 📖 阅读 [API 开发指南](./api-guide) 了解项目架构
2. 🔧 查看 [后端开发](./backend/nodejs-guide) 学习后端技术栈
3. 🎨 浏览 [前端开发](./frontend/react-guide) 掌握前端开发

## 开发环境要求

在开始开发之前，请确保你的环境满足以下要求：

- **Node.js**: >= 18.0.0
- **npm**: >= 9.0.0
- **Git**: >= 2.30.0

:::tip 提示
推荐使用 [nvm](https://github.com/nvm-sh/nvm) 来管理 Node.js 版本。
:::

## 获取帮助

如果在开发过程中遇到问题，可以：

- 查看相关技术文档
- 在团队频道提问
- 提交 Issue 到项目仓库
