# 计费支付

## 计费模式

### 订阅制
- 按月/年订阅
- 固定费用
- 自动续费

### 按量计费
- 按实际使用量
- 后付费
- 每月结算

### 混合模式
- 基础订阅费 + 超额按量
- 适合大客户

## 支付方式

- 在线支付（支付宝/微信）
- 对公转账
- 合同签约

## 发票管理

### 申请发票
1. 进入「账单管理」
2. 选择需要开票的账单
3. 填写发票信息
4. 提交申请

### 发票类型
- 增值税普通发票
- 增值税专用发票（需提供资质）

## 账单查询

在「账单中心」可以查看：
- 历史账单
- 消费明细
- 余额和欠费

## 欠费说明

欠费处理流程：
```
1. 账单到期 → 邮件提醒
2. 逾期 3 天 → 服务降级（只读）
3. 逾期 7 天 → 服务暂停
4. 逾期 30 天 → 数据删除
```

## 常见问题

### Q：如何升级方案？
A：进入「订阅管理」，选择新方案并支付差价。

### Q：可以退款吗？
A：订阅未使用部分可按比例退款。

### Q：发票多久能收到？
A：电子发票即时发送，纸质发票 7 个工作日内邮寄。
