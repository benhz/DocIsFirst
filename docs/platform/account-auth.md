# 账号与认证

## 统一账号体系

### 账号类型
- **员工账号**：使用公司邮箱（@example.com）
- **外部账号**：合作伙伴、客户
- **服务账号**：系统间调用

### 账号生命周期
```
创建 → 激活 → 使用 → 停用 → 删除
```

## 登录方式

### 1. SSO 单点登录（推荐）
- 适用于内部员工
- 一次登录，全平台通用
- 支持自动跳转

**登录流程**：
```
用户 → SSO 登录页 → 认证服务 → 跳转回产品
```

### 2. 账号密码登录
- 适用于外部用户
- 需要注册并验证邮箱

### 3. 第三方登录
- 支持微信、钉钉等
- 首次登录需绑定

## 多因素认证（MFA）

### 启用 MFA
1. 进入「账号设置」
2. 选择「安全设置」
3. 启用「两步验证」
4. 扫描二维码绑定

### MFA 方式
- **TOTP**：Google Authenticator、Microsoft Authenticator
- **短信验证码**
- **邮箱验证码**

## API 认证

### Token 认证
```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.example.com/v1/users
```

### OAuth 2.0
支持以下授权模式：
- Authorization Code（Web 应用）
- Client Credentials（服务间调用）
- Implicit（SPA 应用，不推荐）

详见 [API 认证文档](../api/auth.md)

## 常见问题

### 忘记密码怎么办？
点击登录页的「忘记密码」，通过邮箱重置。

### 如何修改密码？
进入「账号设置」→「安全设置」→「修改密码」

### 账号被锁定怎么办？
连续输错密码 5 次会被锁定 30 分钟，或联系管理员解锁。
