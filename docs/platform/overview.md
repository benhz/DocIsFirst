# 平台与通用能力

这里汇总了跨产品的统一能力，所有产品都复用这些平台级服务。

## 核心能力

### 🔐 [账号与认证](./account-auth.md)
- 统一登录（SSO）
- 多因素认证（MFA）
- OAuth 2.0 / OIDC 支持

### 👥 [权限管理](./permission.md)
- 基于角色的访问控制（RBAC）
- 细粒度权限控制
- 权限审计

### 📢 [消息通知](./notification.md)
- 站内信
- 邮件通知
- 短信 / 企业微信通知

### 💳 [计费支付](./billing.md)
- 订阅管理
- 计量计费
- 发票管理

### 🔗 [集成能力](./integration-overview.md)
- 与外部系统集成
- 数据同步
- Webhook 机制
