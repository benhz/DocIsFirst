# 功能详解

## 智能问答

### 自然语言查询

#### 基础查询
直接用日常语言提问：
- "上个月的销售额是多少？"
- "今天有多少新用户注册？"
- "本季度利润最高的产品是什么？"

#### 复杂分析
支持多维度复杂分析：
- "对比去年同期，今年各区域的销售增长率"
- "按产品类别分析近三个月的销售趋势"
- "找出客户流失率超过 20% 的城市"

#### 对话式分析
支持连续对话，理解上下文：
```
用户: "上个月销售额是多少？"
系统: "上个月销售额为 500 万元"

用户: "环比增长呢？"
系统: "环比增长 15%"

用户: "再看看各区域的表现"
系统: [显示区域销售分布图表]
```

### AI 能力

#### 智能推荐
- **问题推荐**：根据数据和行业推荐常见问题
- **维度推荐**：自动识别关键分析维度
- **指标推荐**：智能推荐相关指标
- **图表推荐**：根据数据特征推荐图表类型

#### 洞察发现
- **异常检测**：自动发现数据异常
- **趋势识别**：识别上升/下降趋势
- **相关性分析**：发现指标间关联
- **原因推断**：分析变化原因

#### 智能提示
- **问题纠错**：自动纠正问题表述
- **语义理解**：理解同义词和业务术语
- **单位转换**：自动处理单位换算
- **时间识别**：智能识别时间表达
