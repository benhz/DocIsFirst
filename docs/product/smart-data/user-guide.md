# 使用手册

## 快速开始

### 连接数据源

#### 添加数据库连接

1. 点击「数据源」→「添加数据源」
2. 选择数据库类型（MySQL、PostgreSQL 等）
3. 填写连接信息：
   ```
   主机地址: 192.168.1.100
   端口: 3306
   数据库名: sales_db
   用户名: analyst
   密码: ********
   ```
4. 点击「测试连接」
5. 连接成功后保存

#### 上传文件

1. 点击「数据源」→「上传文件」
2. 选择 Excel、CSV 或其他文件
3. 预览数据
4. 确认数据类型
5. 完成上传

## 可视化分析

### 创建图表

#### 方法一：从问答创建

1. 在智能问答中提问
2. 查看自动生成的图表
3. 点击「保存图表」
4. 选择保存位置

#### 方法二：手动创建

1. 点击「新建」→「图表」
2. 选择数据集
3. 拖拽字段到相应区域：
   - X轴：拖入维度字段
   - Y轴：拖入指标字段
   - 分组：拖入分类字段
4. 选择图表类型
5. 配置图表样式

### 图表配置

#### 数据配置

**选择指标：**
- 求和、平均值、计数
- 最大值、最小值
- 去重计数

**筛选数据：**
```
销售额 > 10000
区域 = "华东"
日期 在 [2023-01-01, 2023-12-31]
```

**排序：**
- 升序/降序
- Top N
- 自定义排序

#### 样式配置

**颜色设置：**
- 使用预设配色方案
- 自定义颜色
- 按条件设置颜色

**标签设置：**
- 显示数值
- 显示百分比
- 自定义格式：`¥{value}`

**图例设置：**
- 位置：顶部/底部/左侧/右侧
- 样式：横向/纵向
- 显示/隐藏

## 报表制作

### 创建报表

1. 点击「新建」→「报表」
2. 选择模板或空白报表
3. 拖拽组件到画布：
   - 图表组件
   - 文本组件
   - 筛选器
   - 图片组件
4. 调整布局和样式
5. 保存报表

### 使用模板

#### 选择模板

1. 点击「从模板创建」
2. 浏览模板库：
   - 通用模板
   - 行业模板
   - 我的模板
3. 预览模板
4. 选择并创建

#### 自定义模板

1. 创建报表
2. 点击「另存为模板」
3. 填写模板信息
4. 保存到我的模板

### 添加交互

#### 筛选器

1. 拖入「筛选器」组件
2. 选择筛选字段
3. 设置筛选类型：
   - 下拉单选
   - 下拉多选
   - 日期范围
   - 数值范围
4. 关联需要筛选的图表

#### 钻取

1. 选择图表
2. 配置钻取路径：
   ```
   区域 → 城市 → 门店
   ```
3. 设置钻取方式：
   - 点击钻取
   - 右键菜单
4. 保存配置

### 分享报表

#### 生成分享链接

1. 点击「分享」按钮
2. 设置分享权限：
   - 仅查看
   - 可以筛选
3. 设置访问控制：
   - 公开访问
   - 密码保护
   - 仅特定用户
4. 设置有效期
5. 复制链接

#### 定时推送

1. 点击「定时推送」
2. 设置推送计划：
   ```
   频率: 每周一 09:00
   格式: PDF
   收件人: user1@company.com, user2@company.com
   ```
3. 启用推送

## 数据监控

### 创建监控大屏

1. 点击「新建」→「监控大屏」
2. 选择布局模板
3. 添加 KPI 卡片：
   - 拖入 KPI 组件
   - 选择指标
   - 设置显示样式
4. 添加图表
5. 设置自动刷新间隔

### 配置告警

#### 创建告警规则

1. 进入「监控」→「告警设置」
2. 点击「新建告警」
3. 配置告警条件：
   ```
   指标: 日销售额
   条件: 小于
   阈值: 100000
   时间: 实时
   ```
4. 设置告警方式：
   - ✅ 邮件
   - ✅ 企业微信
   - ✅ 短信
5. 设置接收人
6. 启用告警

#### 告警示例

**业务告警：**
```
销售额低于目标值
库存低于安全库存
用户流失率超过预警值
```

**系统告警：**
```
数据同步失败
查询超时
数据源连接异常
```

## 数据管理

### 创建数据集

#### SQL 数据集

1. 点击「数据集」→「新建数据集」
2. 选择数据源
3. 编写 SQL 查询：
   ```sql
   SELECT
     date,
     region,
     product,
     SUM(amount) as total_sales
   FROM sales
   WHERE date >= '2023-01-01'
   GROUP BY date, region, product
   ```
4. 预览数据
5. 保存数据集

#### 可视化建模

1. 选择「可视化建模」
2. 从左侧拖入表
3. 连接表关系
4. 选择字段
5. 添加筛选条件
6. 预览并保存

### 添加计算字段

1. 打开数据集
2. 点击「添加计算字段」
3. 输入字段名和公式：
   ```
   名称: 利润率
   公式: (销售额 - 成本) / 销售额 * 100
   ```
4. 选择数据类型
5. 保存

### 数据同步

#### 设置同步计划

1. 进入数据集设置
2. 点击「数据同步」
3. 选择同步方式：
   - 全量同步
   - 增量同步
4. 设置同步计划：
   ```
   频率: 每天
   时间: 02:00
   超时: 1小时
   ```
5. 启用同步

- 使用数据集而非直连
