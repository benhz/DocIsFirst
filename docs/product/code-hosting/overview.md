# 代码托管平台概述

## 产品简介

代码托管平台是一款面向企业及团队的 Git 代码托管与协作平台，提供从代码版本管理、代码评审，到构建集成与权限控制的全流程能力。  
平台支持私有化部署，确保企业代码资产在受控环境中安全存储，适用于对安全性、可审计性和协作效率要求严格的组织。

平台核心目标是为研发团队提供安全可靠、可控透明、高效协作的现代化开发平台。

## 核心功能概览

以下内容为核心功能模块介绍。你可以在对应功能页面查看更深入说明。

### 1. 仓库管理

- 支持公开 / 私有仓库
- 多级组织结构（组织 / 项目 / 仓库）
- 代码浏览、提交历史、文件 Diff 等常用能力
- 支持基础在线编辑（可按需启用或关闭）

### 2. 代码评审（Merge Request）

- 标准化 MR 工作流
- 行级评论与多次迭代更新
- 支持必选评审人、评审规则、状态追踪
- 变更内容可追溯

### 3. 分支与合并策略

- 支持常见分支模型（Git Flow、Trunk-Based 开发）
- 分支保护策略（禁止直接推送、强制 MR 等）
- 合并策略：  
  - Fast-forward  
  - Merge Commit  
  - Squash Merge  
- 自动冲突检测并提示处理

### 4. 权限与安全管理

- 基于 RBAC 的权限模型
- 仓库级、项目级、分支级权限细分
- 完整的审计日志与活动记录
- 支持 SSH Key、Token 等多种访问方式
- 可与组织现有身份系统集成（LDAP/SSO）

### 5. CI/CD 集成能力

- 支持 Webhook 触发外部构建系统
- 构建与测试状态展示在提交/MR 中
- 支持查看构建日志、重试构建等能力（视集成方式而定）
- 可与 Jenkins、GitLab CI、GitHub Actions 等系统对接

### 6. 数据统计与分析

- 仓库提交趋势
- Merge Request 活跃度
- 团队贡献分布
- 项目活动历史与健康度趋势
