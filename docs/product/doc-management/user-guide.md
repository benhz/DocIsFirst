# 使用手册

## 快速开始

### 创建空间

1. 点击「创建空间」
2. 选择空间类型：
   - 个人空间：私人使用
   - 团队空间：团队协作
   - 项目空间：项目文档
3. 填写空间信息：
   ```
   名称: 产品团队
   描述: 产品相关文档
   可见性: 仅成员
   ```
4. 创建完成

### 创建文档

#### 新建空白文档

1. 进入空间
2. 点击「新建文档」
3. 选择文档类型：
   - Markdown 文档
   - 富文本文档
   - 表格文档
4. 输入标题开始编辑

#### 从模板创建

1. 点击「从模板创建」
2. 选择模板
3. 填充内容
4. 保存

### 邀请成员

1. 进入空间设置
2. 点击「邀请成员」
3. 输入邮箱地址
4. 选择角色权限
5. 发送邀请

## 文档编辑

### 基础操作

#### 文本格式化

**工具栏：**
- 标题：H1-H6
- 加粗：Ctrl+B
- 斜体：Ctrl+I
- 列表：Ctrl+Shift+7/8
- 链接：Ctrl+K

#### 插入元素

**图片：**
```
方式1: 拖拽图片到编辑器
方式2: 点击图片按钮上传
方式3: 粘贴剪贴板图片
方式4: 插入图片URL
```

**代码块：**
```
输入 ``` 选择语言
或点击代码块按钮
```

**表格：**
```
点击表格按钮 → 选择行列数 → 开始编辑
```

### 协作编辑

#### 实时协作

**开始协作：**
1. 打开文档
2. 邀请协作者
3. 同时编辑

**协作提示：**
- 看到其他用户光标
- 实时同步更改
- 避免同时编辑同一段落

#### 添加评论

**行内评论：**
1. 选中文本
2. 点击评论按钮
3. 输入评论内容
4. @提及相关人员
5. 提交评论

**回复评论：**
```
评论 → 点击回复 → 输入内容 → 提交
```

#### 解决评论

```
查看评论 → 处理问题 → 标记为已解决
```

## 文档管理

### 组织结构

#### 创建目录

1. 点击「新建目录」
2. 输入目录名称
3. 拖拽文档到目录

#### 移动文档

**方式1: 拖拽**
```
拖动文档 → 放到目标位置
```

**方式2: 菜单**
```
右键文档 → 移动 → 选择目标位置
```

### 标签管理

#### 添加标签

```
打开文档 → 点击标签 → 输入标签名 → 回车
```

**标签示例：**
```
#产品 #需求 #v2.0 #高优先级
```

#### 标签筛选

```
点击标签 → 查看该标签下所有文档
```

### 搜索文档

#### 快速搜索

```
Ctrl+K → 输入关键词 → 选择文档
```

#### 高级搜索

1. 点击搜索框
2. 点击「高级搜索」
3. 设置筛选条件：
   ```
   关键词: 产品需求
   创建人: 张三
   时间范围: 最近30天
   标签: #产品
   ```
4. 查看结果

## 版本管理

### 查看版本历史

1. 打开文档
2. 点击「版本历史」
3. 查看版本列表

### 对比版本

1. 选择两个版本
2. 点击「对比」
3. 查看差异：
   - 🟢 新增内容
   - 🔴 删除内容
   - 🟡 修改内容

### 恢复版本

**方法1: 回滚**
```
选择版本 → 预览 → 回滚到此版本
```

**方法2: 创建副本**
```
选择版本 → 创建副本 → 新文档
```

## 权限设置

### 设置文档权限

1. 打开文档
2. 点击「分享」
3. 设置权限：
   ```
   成员1: 可编辑
   成员2: 仅查看
   其他人: 无权限
   ```

### 创建分享链接

1. 点击「创建分享链接」
2. 配置选项：
   ```
   权限: 仅查看
   需要密码: 是
   有效期: 7天
   ```
3. 复制链接

## 模板使用

### 使用预置模板

1. 新建文档
2. 选择「从模板创建」
3. 浏览模板库
4. 选择合适的模板
5. 开始编辑

### 创建自定义模板

1. 创建并编辑文档
2. 点击「...」→「另存为模板」
3. 设置模板信息：
   ```
   名称: 产品需求文档模板
   分类: 产品
   描述: 标准PRD模板
   ```
4. 保存

## 导入导出

### 导入文档

**上传文件：**
1. 点击「导入」
2. 选择文件（支持批量）
3. 选择导入位置
4. 开始导入

**支持格式：**
- .docx, .doc
- .md
- .html
- .pdf (仅查看)

### 导出文档

1. 打开文档
2. 点击「导出」
3. 选择格式：
   - PDF
   - Word
   - Markdown
   - HTML
4. 下载文件

## 最佳实践

### 文档命名

**规范建议：**
```
好的命名:
- PRD-用户登录功能-v1.0
- 2024年产品路线图
- API接口文档

避免:
- 新文档
- 未命名文档
- 文档1、文档2
```

### 目录组织

**推荐结构：**
```
项目空间/
├─ 00-项目管理/
│   ├─ 项目计划
│   └─ 会议纪要
├─ 01-需求文档/
├─ 02-设计文档/
├─ 03-开发文档/
└─ 04-测试文档/
```

### 版本管理

**重要版本标记：**
```
v1.0 - 初始版本
v1.1 - 评审通过版本
v2.0 - 大版本更新
```

### 权限管理

**权限原则：**
- 最小权限原则
- 定期审查权限
- 敏感文档单独授权
- 离职及时回收

## 故障排查

### 文档无法保存

**检查：**
1. 网络连接
2. 存储空间
3. 权限设置

### 协作冲突

**解决：**
```
发现冲突 → 查看冲突内容 → 手动合并 → 保存
```

### 文档丢失

**恢复：**
1. 检查回收站
2. 查看版本历史
3. 联系管理员
