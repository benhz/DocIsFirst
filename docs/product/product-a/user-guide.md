## 常见操作

### 批量导入数据
1. 准备数据文件（CSV 格式）
2. 点击「导入」按钮
3. 选择文件并上传
4. 检查导入结果

### 导出报表
1. 选择导出时间范围
2. 选择导出字段
3. 选择导出格式（Excel/CSV）
4. 点击「导出」
