# 产品中心

欢迎来到产品中心！这里汇总了我们为开发者和企业提供的四大核心平台产品。

## 产品总览

### [代码托管平台](./code-hosting/overview.md) 💻
**一句话介绍**：企业级代码托管与协作平台，提供安全可靠的代码版本管理能力

**核心功能**：
- Git 仓库管理与权限控制
- 代码评审与合并请求
- 分支保护与代码规范检查
- CI/CD 工具链集成

**适用场景**：
- 企业代码资产统一管理
- 团队协作开发与代码评审
- DevOps 流程自动化
- 开源项目托管

## 如何选择产品？

| 需求场景 | 推荐产品 |
|---------|---------|
| 代码版本管理与团队协作 | 代码托管平台 |
| 数据分析与可视化 | 智慧问数平台 |
| 构建AI智能助手 | 智联知识库智能体开发平台 |
| 文档管理与知识沉淀 | 文档管理平台 |
| 完整的研发协作体系 | 代码托管 + 文档管理 |
| 智能化知识服务 | 知识库智能体 + 文档管理 |

## 产品集成

我们的产品支持灵活集成，可以根据您的业务需求组合使用：

- **研发协作套件**：代码托管平台 + 文档管理平台
- **数据智能套件**：智慧问数平台 + 智联知识库智能体开发平台
- **全栈开发者平台**：四大平台完整集成
