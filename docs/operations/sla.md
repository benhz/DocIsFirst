# SLA 与值班制度

## SLA 承诺

### 可用性
- 核心服务：99.95%（月停机 < 22 分钟）
- 一般服务：99.9%（月停机 < 45 分钟）

### 响应时间
- P95 < 1s
- P99 < 3s

### 错误率
- < 0.1%

## 值班制度

### 值班时间
- 工作日：9:00 - 21:00（主值班）
- 工作日：21:00 - 次日 9:00（备值班）
- 周末/节假日：24 小时值班

### 值班职责
- 监控告警响应
- 故障处理
- 发布支持

### 值班排期
每周五公布下周排期，在企业微信群内通知。

## 考核指标

- 告警响应时间
- 故障处理时长
- SLA 达成率
